    Top Chef Website

    Le but de ce site web est de trouver tous les restaurants étoilés Michelin qui offrent des promotions (en France).
    "Eat well and cheaper than usually".

    Pour cela, j'ai "scrapé" le site web Michelin pour récupérer une liste de tous les restaurants étoilés en France (en utilisant Node.js et le module cheerio).
    Ensuite, j'ai cherché ces restaurants sur lafourchette.com en utilisant leur nom et leur adresse, afin d'être sûr qu'il s'agissait bien du même restaurant. Enfin, j'ai retourné une nouvelle liste de restaurants avec toutes les promotions existantes.

    Pour la partie "front", j'ai utilisé React sur mon site web pour afficher les restaurants et les promotions.
    L'utilisateur peut filtrer les restaurants en indiquant le nombre d'étoiles Michelin.

    Développement d'une application Android

    But


    Le but de ce projet était de développer une application Android similaire à Youtube, en utilisant leur API.


    Fonctionnalités

    • Recherche de vidéos YouTube: vous pouvez faire des requêtes en rentrant du texte dans la searchview, une liste de vidéos sera affichée dans un RecyclerView implémentant une CardView pour chaque vidéo.
    • Historique: toutes les précédentes recherches sont sauvegardées même si l'application est fermée. L'utilisateur est assisté par l'autocomplétion.
    • Afficher les vidéos: appuyez sur les vidéos pour obtenir une nouvelle vue plus détaillée, où vous pourrez jouer la vidéo grâce à l'API YouTube Android Player.
    • Bouton d'accueil: appuyer sur le bouton "home" dans l'appbar pour afficher les vidéos YouTube populaires du moment.

    • Date: Mars 2018
    • GitHub repository
